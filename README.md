# 리니지 변신 리스트 수정 도구

Win32 C++로 만든 변신 리스트(`.e` / `.spz.e` 등) 편집·변환·검증 도구입니다.  
Windows 7 이상에서 동작합니다.

제작: Light · 지원: 갓!소피아

## 빌드

요구 사항: CMake 3.15+, Visual Studio(MSVC), Windows SDK

```bat
build.bat
```

실행 파일: `build\Release\변신리스트수정 툴.exe`

배포용(정적 런타임 `/MT`)은 `build_dist.bat`을 사용합니다.

## 주요 기능

| 메뉴 | 설명 |
|------|------|
| 파일 읽기 / 저장 | UTF-8(CRLF) 텍스트 열기·저장. 최대 **128MB** |
| 들여쓰기 변환 | 빈 줄·불필요 공백 정리 |
| 기호 변환 | 괄호·구두점 등 제거 (액션명 유지). `N.(…)` 형식 작업 시에는 보통 사용하지 않음 |
| 영문자 삭제 | `#` 헤더 영문명·데이터 줄 액션명(`walk`, `effect` 등) 전부 제거 → `N.(…)` 형식 |
| 키워드 찾기 | 검색 전체 / 위로 / 아래로 / 지우기 · **F3** · **Shift+F3** |
| 순서 검증 | `#0`, `#1`, `#2` … 연속 번호 확인 |

## 영문자 삭제 결과 형식

원본:

```text
#0	312=3225	prince
	0.walk(1 4,0.0:4<479 0.1:4 0.2:4<478 0.3:4)
	4.walk onehandsword(1 4,104.0:4<479 104.1:4 104.2:4<478 104.3:4)
```

변환 후:

```text
#0	312=3225
	0.(1 4,0.0:4<479 0.1:4 0.2:4<478 0.3:4)
	4.(1 4,104.0:4<479 104.1:4 104.2:4<478 104.3:4)
```

규칙 요약:

- `#` 줄: 영문 이름만 제거 (`#1186 312=3221 sorceress` → `#1186 312=3221`)
- 데이터 줄: `A–Z` / `a–z` 삭제, 괄호·숫자·`<` `>` `[` `]` `!` `:` `,` `.` 유지
- 액션명 띄어쓰기로 생기던 `4. (` 공백 제거 → `4.(` 로 통일
- 선행 탭 들여쓰기 유지

권장 순서: 원본 파일을 연 뒤 **영문자 삭제**만 실행합니다.  
이 형식에서는 **기호 변환**을 쓰지 않는 것이 안전합니다(괄호·콜론 등이 깨질 수 있음).

## 수정 내용 (변경 이력)

### 2026-07

- **영문자 삭제**를 `#` 헤더 + 데이터 줄(`0.walk`, `0.effect` 등) 전부 대상으로 통일
- `4.walk onehandsword(` → `4.(` 처럼 **점과 `(` 사이 공백** 제거
- `0,walk(` 형태는 `0.(` 로 보정
- 데이터 줄 앞 **탭** 유지 (공백 축약이 탭을 지우지 않도록 수정)
- **찾기** 창: 검색 전체 / 위로 / 아래로 / 지우기, F3 / Shift+F3
- 파일 크기 제한 **16MB → 128MB**, RichEdit 입력 한도 상향
- 잘못 삭제되던 기호 변환·영문 삭제(액션명까지 깨뜨리던 방식) 롤백 후, SPR에 맞는 `N.(…)` 형식으로 재구현
- GitHub 공개: https://github.com/firstluffy/lineage-list-tool

### 알려진 주의사항

- 영문 삭제 후 `0.effect` → `0.` 형태가 됩니다. 뷰어/툴이 이 문법을 요구하는지 확인하세요.
- `#1186` SPR 오류는 과거에 액션명까지 잘못 지우거나 기호 변환으로 구조를 깨뜨린 경우와 관련이 있었습니다. 원본에서 **영문자 삭제만** 다시 적용하는 것을 권장합니다.
